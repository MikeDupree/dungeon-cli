# Dungeon crawl CLI

## Introduction

## Requirements
- pnpm
- ts-node
- a command line

## How to play

- clone the repo
- `pnpm i` to install the dependencies
- run `./index.ts`
  > You may need to add executable permissions. 
  > If you don't want to do that just run `ts-node index.ts`
